Design

Adapt-text-for-the-web

Write short texts in a direct style. Express ideas as concisely as possible, even if this means spreading the text across several pages (if it is long and/or incorporates wide-ranging thoughts). Split lengthy content into several pages.

If a topic requires a long explanation, split it into several pages and/or put it in a downloadable document. This way, users will only display pages that they wish to read, and only download files that interest them.
