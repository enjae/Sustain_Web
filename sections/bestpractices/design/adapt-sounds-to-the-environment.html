Design

Adapt-sounds-to-the-environment

Audio files can be large and use up lots of bandwidth. It is therefore important to minimize their size. Choose one of the 3 formats that cover the 3 largest platforms (Windows, Mac OS X and Linux):
- MP3 (MPEG-1 Audio Layer 3) - AAC (Advanced Audio Coding) - Vorbis

These formats apply fine-tuned compression algorithms that greatly reduce file size.

Encoders like LAME not only convert uncompressed audio files into MP3, but also change the sampling rate to save even more space, at the cost of poorer audio quality. You should therefore test it on each sound file. A large son.wav WAV file of 63,128 bytes converted to MP3 produces: - a son-128.mp3 file of 10,823 bytes (128 kb/s), 6 times lighter - a son-64.mp3 file of 6,508 bytes (64 kb/s), 10 times lighter
